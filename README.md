# HeadHunter API

Получение информации о вакансиях при помощи api от сервиса HeadHunter.



## Зависимости

Стандартная библиотека `requests`
    
## Запуск

Запуск скрипт `hh_api.py` и отвечаем на вопросы

```bash
  python3 hh_api.py 
```


## Особенности

- Получение списка вакансий прямо в терминале
